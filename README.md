# 📅 スケジュール管理アプリ

カレンダー形式で予定を管理できるWebアプリです。React + Vite で作っています。

## 機能

- **月表示カレンダー** — 前月・翌月への移動、「今日」ボタンで今月に戻る
- **予定の追加・編集・削除** — 日付マスをクリックするとその日の予定一覧が開き、「＋予定を追加」で同じ日に何個でも予定を登録できる。予定クリックで編集・削除
- **時間帯の表示** — 開始・終了時間を任意で設定でき、「18:00〜21:00」のように表示される
- **カテゴリによる色分け** — 授業・バイト・副業・就活・プライベートなどのカテゴリごとに文字色が変わり、視覚的に見やすい
- **目標リスト** — 「◯月◯日までに達成したいこと」を登録でき、残り日数のカウントダウン付き。締切日はカレンダー上にも 🎯 で表示される
- **中間ステップ（逆算プランニング）** — 目標を「デザイン決め: 7/12まで」のような中間ステップに分解できる。各ステップはカレンダーの締切日に ⚐ で表示され、チェックで達成管理
- **時間の使いみち集計** — カテゴリ別の予定時間を今週／今月で集計し、円グラフで表示。時間未設定の予定は2時間として数える
- **空き時間の見える化** — 活動時間（9:00〜24:00、`src/config.js` で変更可）のうち予定が入っていない時間帯を、日付クリック時のモーダルにリスト表示
- **フリーメモ** — カレンダーとは無関係に、なんでも自由に書き込めるメモ欄。入力内容は自動保存される
- **データの自動保存** — 入力内容はブラウザの localStorage に保存されるため、ブラウザを閉じたり開発サーバーを止めてもデータは消えない

- **AI機能（任意・従量課金）** — ✨ボタンから自然な文章で予定を一括登録したり、目標の中間ステップをAIに提案してもらえる。Claude API（Sonnet 5）を使用し、1回約2〜3円。利用回数と実測コストはアプリ内の「🤖 AI利用状況」パネルで月予算（300円）と比較できる。**手入力は今まで通り何回でも0円**

## 使い方

```bash
# 依存パッケージのインストール（初回のみ）
npm install

# 開発サーバーの起動（アプリ本体 + AI用サーバーが同時に起動する）
npm run dev
```

ブラウザで http://localhost:5173 を開くとアプリが表示されます。

### AI機能を使う場合（任意）

AI機能を使うには Anthropic のAPIキーが必要です。

1. `.env.example` をコピーして `.env` というファイルを作る
2. `ANTHROPIC_API_KEY=sk-ant-...` に自分のキーを貼り付ける
3. `npm run dev` を起動し直す

`.env` は `.gitignore` に登録済みなので、GitHubにキーが上がることはありません。
キーを設定しなくてもアプリ本体（手入力）は全機能そのまま使えます。

## 技術スタック

- [React](https://react.dev/) — UIライブラリ
- [Vite](https://vite.dev/) — ビルドツール・開発サーバー
- localStorage — データの永続化（サーバー・DB不要）

## ディレクトリ構成

```
src/
├── App.jsx                    # アプリ全体のレイアウトと状態管理
├── categories.js              # 予定カテゴリと色の定義
├── utils.js                   # 日付計算などのヘルパー関数
├── hooks/
│   └── useLocalStorage.js     # localStorage への自動保存フック
└── components/
    ├── Calendar.jsx           # 月表示カレンダー
    ├── EventModal.jsx         # 予定の追加・編集モーダル
    └── GoalPanel.jsx          # 目標リストのパネル
```
